# Priyansh Pratap Portfolio

A premium personal portfolio showcasing my software engineering, AI, Flutter, and full-stack development projects.

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lenis
- Lucide Icons

## ✨ Features

- Premium editorial UI
- Responsive design
- Smooth animations
- Interactive project showcase
- Resume download
- Contact section
- Social media integration

## 🛠️ Installation

```bash
npm install
npm run dev
```
